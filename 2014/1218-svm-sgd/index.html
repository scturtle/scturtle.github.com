from sklearn.datasets import load_iris
from sklearn.preprocessing import scale
data = load_iris()
X0, y0 = data.data, data.target
y = y0[y0 != 2]
y[y == 0] = -1
X = X0[y0 != 2, :]
X = scale(X[:, [2, 3]])
X = np.c_[X, np.ones(X.shape[0])]

SVM can be writen as:

$$ \min_w \frac 1 n \sum_{i=1}^n \max\{0, 1-y_i \langle w, X_i \rangle \} + \frac \lambda 2 \|w\|^2 $$

A form of Hinge loss + penalty.

Average for each $X_i$:
$f(w;i) = \max\{0, 1-y_i \langle w, X_i \rangle \} + \frac \lambda 2 \|w\|^2$

The derivative:
$\nabla_i = \lambda w - [y_i \langle w, X_i \rangle < 1] y_i X_i$

Update rule:
$w \leftarrow (1 - \frac 1 i)w + \frac 1 {\lambda \cdot i} [y_i \langle w, X_i \rangle < 1] y_i X_i $

Ref: [Shai Shalev-Shwartz, et al., 2011]

n, p = X.shape
w = np.zeros(p)
l = 0.01
T = 10*n
for t in xrange(1, T+1):
    i = np.random.randint(n)
    w = (1.-1./t) * w
    if y[i]*w.dot(X[i]) < 1:
        w += y[i]*X[i]/l/t
print 'weight:', w
print 'err_cnt:', np.sum(w.dot(X.T)*y < 0)
